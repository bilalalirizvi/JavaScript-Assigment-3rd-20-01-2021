Question Number 2

Generate the following series using for loop.


Counting




Here is a Code!


            for (var a = 1; a <= 15; a++) {
                document.write(a + ",");
            }

Reverse Counting




Here is a Code!


            for (var b = 15; b >= 1; b--) {
                document.write(b + ",");
            }

Even




Here is a Code!


            for (var c = 0; c <= 20; c = c + 2) {
                document.write(c + ",");
            }

Odd




Here is a Code!


            for (var d = 1; d <= 20; d = d + 2) {
                document.write(d + ",");
            }
