Question Number 4

Initialize an array with color names (Array length should be 5). Log that array in your console

HINT: push, pop, shift, unshift, splice
  1. Ask the user what color he/she wants to add to the beginning & add that color to the beginning of the array. Display the updated array values in your browser using Alert (All the values should be listed)
  2. Ask the user what color he/she wants to add to the end & add that color to the end of the array. Display the updated array in your browser using Alert (All the values should be listed)
  3. Add two more color to the beginning of the array. Display the updated array in your browser using Alert (All the values should be listed)
  4. Delete the first color in the array. Display the updated array in your browser using Alert (All the values should be listed)
  5. Delete the last color in the array. Display the updated array in your browser using Alert (All the values should be listed)
  6. Ask the user at which index he/she wants to add a color & color name. Then add the color to desired position/index. Display the updated array in your browser using Alert (All the values should be listed)
  7. Ask the user at which index he/she wants to delete color(s) & how many colors he/she wants to delete. Then remove the same number of color(s) from user-defined position/index. . Display the updated array in your browser using Alert (All the values should be listed)
