Question Number 1

Write a program to sum the first 100 numbers using for loop

Expected Output: 5050


Here is a Code!


        var sum = 0;

        for (var i = 1; i <= 100; i++) {
            sum += i;
        }

        alert("The Sum of Numbers: " + sum);
